Начнем со стандартных команд для получения информации о пользователе:
1. whoami - имя пользователя на данный момент 
2. id - для вывода большей части инфы о пользователе 
3. w - просмотр(какие пользователи в сессии)

Далее команды для создания пользователей и работы с ними:
	1. adduser - создаем юзера и сразу помещаем в /home каталог, что реально удобно
	2. useradd bebe1 -m - тоже самое но с ключом -m не забыть
	3. useradd bebe1 -m -s  /home/bebefu1 - ключом -s задаем каталог в домашней папке, то есть по сути должно было быть что-то типо /home/bebe1, если бы писали без ключа -s
	4. sudo cat /etc/shadow - посмотреть хеши паролей и прочую инфу про пользователей(самых разных, в том числе и user и daemon и root)
	5. sudo cat /etc/passwd - что-то похожее с другим содерж
	6. можно кстати командой passwd изменять пароль у пользователей, пишем passwd (имя пользователя) и меняем пароль
	7. Кстати если у пользователя nologin - это демон, типо процесс, который в фоновом режиме работает)
	8. usermod - полезная команда с ключами(в целом дополняет инфу о пользователе
Допустим хотим выдать пользователю sudo группу, чтобы он типо был суперпользователем, ну окей, пишем usermod -a -G bebe1 таким образом при вводе: groups у него будет sudo группа
	9. usermod -L - заблокировать пользователя
	10. usermod -U - разблокировать пользователя
	11. Кстати когда ты блокируешь пользователя, в хеше пароля просто ставится ! знак и всё!
	12. addgroup - добавить в группу пользователя с ключом -a -G 
	13. sudo delusers ivan_tsar admins - удалить из группы
