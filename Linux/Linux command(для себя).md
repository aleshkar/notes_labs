Итак, приступим к изучению команд для работы с Linux:
По сути их можно разделить на множество блоков:

1. Начнем с лайфхаков для удобств в использовании: 
Горячие клавиши: 
Ctrl+U - быстро стереть текст(строчку)
Ctrl+I - стереть все, но можно подняться наверх
2. clear - команда, которая стирает всё в консоли
3. а вот команда history - поможет посмотреть историю команд которую ты ввёл 
4. history -c - полностью очистит историю 
5. если же нужно исполнить определенную команду из history, а тебе лень её копировать или переписывать, то можно 

wget ... - скачать что-то из инета
less -p - проверка
